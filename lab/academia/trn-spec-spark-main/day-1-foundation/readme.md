## connecting locally with [pyspark] with pycharm

```sh
# installing apache spark
brew install apache-spark
brew reinstall apache-spark

# verify version
pyspark --version

# open spark-shell & spark-submit
# /Users/luanmorenomaciel
spark-shell
spark-submit
```
